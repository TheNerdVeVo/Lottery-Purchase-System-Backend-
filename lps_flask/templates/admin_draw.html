📌 Demo Tip

To showcase the full winner flow to a grader:

  1. Register a test customer account
  2. Login as them and buy a ticket with manual numbers, e.g. 1,2,3,4,5
  3. Logout, login as admin (this page)
  4. Enter 1,2,3,4,5 in the Force Numbers field for that game and run the draw
  5. Login back as the test customer — they'll get a winner notification, and the order page will show "Claim Prize"
  6. Click claim — winnings are credited to the customer's wallet
